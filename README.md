# Homebrew Tap

This is a Homebrew tap for [Codive](https://github.com/touseefliaqat/agent-rust) - an AI coding agent CLI built in Rust.

## Installation

```bash
brew tap touseefliaqat/tap
brew install codive
```

## Update

```bash
brew update
brew upgrade codive
```

## Uninstall

```bash
brew uninstall codive
brew untap touseefliaqat/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| codive  | AI coding agent CLI built in Rust |
